# Domain Controller Setup

## Overview
Document the installation and configuration of a Windows Server
Domain Controller for an Active Directory lab.

## Environment
- Windows Server 2019/2022
- Active Directory Server Manager

## Steps Performed
- Installed Windows Server 2022
- Set a static IP
- Installed AD DS
- Promoted server to Domain Controller
- Created a new domain

## Result
Domain Controller is operational and managing domain services.
